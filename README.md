# Isa's Version 🎵

Player de áudio personalizado com tema romântico rosa e branco, desenvolvido com Next.js e TypeScript.

![Isa's Version](https://img.shields.io/badge/version-1.0.0-FF6B9D)
![Next.js](https://img.shields.io/badge/Next.js-14.0.4-FF6B9D)
![TypeScript](https://img.shields.io/badge/TypeScript-5.3.3-FF6B9D)

## ✨ Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| Play/Pause | Controle central com animação |
| Manipular volume | Slider com indicador percentual |
| Barra de progresso | Clique para navegar na música |
| Avançar +10s | Pula 10 segundos na música |
| Retroceder -10s | Volta 10 segundos na música |
| Próxima música | Avança para a próxima faixa |
| Música anterior | Volta para a faixa anterior |
| Playlist | Lista clicável com todas as músicas |
| Destaque visual | Música atual em destaque com bolinha pulsante |

## 🎨 Design

- **Cores:** Tons de rosa (#FF6B9D, #FFB8D0, #FF3388) e branco
- **Fontes:** Playfair Display (títulos) + Poppins (textos)
- **Animações:** Corações flutuantes, glow suave, pulsação
- **Estilo:** Romântico, minimalista, inspirado no estilo Kwai
